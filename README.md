# Global Cyber-Attacks-by-Industry-Sector (2015–2024)

## 📌 Project Overview

This project analyzes global cybersecurity incidents from 2015 to 2024 using Python. It explores cyber attack trends, target industries, financial losses, defense mechanisms, and incident resolution through data visualization.

## 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

## 📊 Visualizations

* Correlation Heatmap
* Attack Type Distribution
* Target Industry Distribution
* Cyber Attack Trend Over Years
* Financial Loss by Industry
* Financial Loss by Country
* Attack Type vs Target Industry Heatmap
* Defense Mechanisms Used
* Incident Resolution Time Distribution
